　　“人不在我这里。”国王冷静的开口，随即冷斥道：“一有点事就冲进来不去看守屋外，是故意要打扰我休息吗？”
<br/>　　一干人连忙又跪在地上，惶恐不已。
<br/>　　“都出去，没有我的允许不许进来。”
<br/>　　“是！”
<br/>　　当人走了个干净，楚涵这才钻出来，立即发问：“说说清楚，白允儿怎么会在天运国呆了二十年？”
<br/>　　国王表情古怪：“她离开这么久你不知道？不是你的妻子吗？”
<br/>　　楚涵心想这事解释不清，只能硬着头皮说道：“未婚妻啊！”
<br/>　　国王不再怀疑，缓缓将事情的发展告知楚涵。
<br/>　　原来天运国在二十年前并不是像现在这般昌盛，只是一个非常不起眼的小国家，一切的改变都是从白允儿出现开始。
<br/>　　她的第一步，就是救了国王，紧接着便任命天运国天女！
<br/>　　白允儿已经在天运国当了二十年的天女，这期间她发现了天运国铺地用的石砖颜色泛黄，好巧不巧正是星玉！
<br/>　　与珈蓝国的情况类似，只不过珈蓝国只是被太子巧合撞见了一个星玉矿区，天运国这里才是真正盛产星玉的地方。
<br/>　　几乎满地都是，由于太多，直接被天运国的人当成了地砖。
<br/>　　发现这一情况后不得了，白允儿当即展开了一个惊天动地的计划。
<br/>　　她要造飞船！
<br/>　　由于救过国王的命又武力值高过国师，白允儿便被任命为天运国天女，专门负责制造飞船。
<br/>　　这么一造就造了足足二十年，甚至白允儿还将信息大爆炸世代的先进思想传递给国王，让她更好的打理国家。
<br/>　　连练兵白允儿都有所涉及，将一部分狼牙的训练模式带到了天运国，让天运国的武力划分简单粗暴又十分有效。
<br/>　　楚涵刚来是看到的那些不同颜色的别针，也正是白允儿想出来的方式，还真附和她喜欢极简的风格。
<br/>　　另外楚涵也恍然大悟，难怪他刚来时就感觉那些普通士兵就跟被洗脑了一样，搞了半天还真是白允儿洗的脑。
<br/>　　紧接着，天运国便在这二十年间迅猛发展，一跃超过这片大陆上的所有国家，成为真正的领军者。
<br/>　　飞船也在一年前造了出来，经过好几轮的测试之后，终于在今天白天正式起航！
<br/>　　听完这些，楚涵已经震惊的无话可说，白允儿真特么牛逼……
<br/>　　为了回地球，竟然在天运国搞出这么大的名堂！
<br/>　　不过要不要这么巧？
<br/>　　白允儿前脚刚走，他就因为异度空间点的回炉碎片，而来到了天运国！
<br/>　　这是命运的安排，还是缘分的使然？
<br/>　　看到楚涵坐在床边一愣就是良久，国王也趁其不备赶紧将衣服穿上，而后强行压下心中的异样开口：“怎么样，回过神没有？”
<br/>　　“没有。”楚涵老是回答，然后继续思考。
<br/>　　国王一呆，被楚涵这种干脆利落的性格镇住，这么不走寻常路的吗？
<br/>　　事情的顺序楚涵已经理清楚了，就是这天运国的时间概念，他搞不清楚。
<br/>　　白允儿不过就是在地球上消失了几个月而已，在这天运国，竟然就过去了二十年！
<br/>　　也不知道她这二十年怎么熬过来的……
<br/>　　“因为磁场时空以及恒星不一样。”这时候的旺财声音响起，缓缓而道：“并不是所有居住地都像地球一样的时间法则。”
<br/>　　“你醒了？！”楚涵惊喜。
<br/>　　旺财的声音透露着许些疲惫：“感应到了回炉碎片在这片大陆，不过你还真是运气好，一来就遇上了白允儿在这里的闺蜜，竟然还是国王。”
<br/>　　关于这一点，楚涵也惊讶的不行，闺蜜这种东西，在末世纪元里他听都没听说过。
<br/>　　事情顺好了之后，楚涵就抬眼对国王道：“我来天运国是找一件东西，找到后就会离开，不过既然你是白允儿的朋友，有什么困难可以直接找我，一定尽力，不过你都是国王了，应该没困难。”
<br/>　　说罢，楚涵便直接打开窗户，一下子就蹿了出去！
<br/>　　国王喊都来不及，只能愣愣的望着楚涵消失不见。
<br/>　　“我连你叫什么名字都不知道啊……”
<br/>　　离开后的楚涵无处可去，随意的找了个不显眼的树杈上躺着，这时候天已经微微亮，不知不觉他竟然和国王聊了一整夜。
<br/>　　他忽然想到之前在珈蓝国的经历，便拿出一个小钱袋。
<br/>　　这是珈蓝国那个将军孙女硬塞给他的，拿到的时候楚涵随手一扔次元空间当成了纪念品，没想到现在又需要拿出来。
<br/>　　珈蓝国和天运国在同一片大路上，货币是一样的吧？
<br/>　　于是天大亮后，楚涵来到了大街上，他的穿着与天运国的人一样，并未有引起注意。
<br/>　　只是满城都有最新的告示张贴在各处，捉拿昨天闯入王宫的人，还有肖像，只不过这个肖像画的跟他本人差距很大。
<br/>　　毕竟他来去速度太快，那帮人跟本没来得及观察。
<br/>　　经过细微的观察过后，楚涵也确认手中的钱袋有用，珈蓝国和天运国的货币还真是同一种。
<br/>　　于是他先找到一处酒馆打算安顿自己，只不过他刚打开钱袋，表情就一下子变了。
<br/>　　莫千伊给他的钱袋里，不是钱！
<br/>　　“我特么……”楚涵表情那叫一个气。
<br/>　　旺财也无语了半天，最后出主意：“去问国王借钱呗，你就只认识她一个。”
<br/>　　于是楚涵只能再次回到树杈上，等待天黑，大白天的容易被发现，还是晚上方便。
<br/>　　只不过昨天刚跟人说有困难会帮忙，没想到今天就主动有事相求。
<br/>　　闲来无事楚涵也翻起了那个钱包，里面一颗颗的也不知道放的什么，像是纽扣又像是宝石。
<br/>　　楚涵完全不懂，总不能再去一趟珈蓝国问吧？
<br/>　　“定情信物。”旺财却弩定。
<br/>　　楚涵沉默了一下，忽然才觉得莫千伊长的挺好看，如果是定情信物的话，那等他忙完了也不是不能再去一趟珈蓝国，反正他的回炉系统开启了通道。
<br/>　　只不过，积分消耗过大！
<br/>　　不再呼吸乱想，楚涵一直安静的等到太阳落山，这期间他也沉得住气，待在树杈上收敛气息。
<br/>　　天黑之后，楚涵整个人也灵活了起来，动身直奔王宫。
