　　萧逸快速赶着路，除了遇到山贼匪窝，需要完成任务外，再不作任何停留。
<br/>　　毕竟，去裂天剑派才是正事。
<br/>　　时间荏苒，两月时间悄然便过。
<br/>　　百武城城外，某处山林中，一少年淡淡地从打坐中醒来。
<br/>　　少年戴着个面具，看不清模样，但眼睛灵动，眉额白哲清秀，想来是个俊俏的小哥儿。
<br/>　　只是，少年身上那锋芒毕露的气息，无不表明，他是个强大的武者。
<br/>　　少年正是萧逸。
<br/>　　这段时间，赶路之余，歇息之时，他会习惯地打坐修炼，要么练一下武技，无聊时也看一看那本清风剑罡。
<br/>　　两月时间，他经历了无数场战斗，和人斗，和妖兽斗，相当于经历了一场历练。
<br/>　　这般高强度的战斗，也让他的修为有所增加，从先天一重达到了先天三重。
<br/>　　仅仅两月，便在先天境中连跳两重，而且大部分时间都是放在赶路，这样恐怖的修炼速度，着实惊人。
<br/>　　要知道，一般的先天武者，每突破一重都是需要极久时间的。
<br/>　　像之前萧家的长老，起码都要数年时间才能突破一重。
<br/>　　如今的萧逸，使用升龙后，已是先天九重战力。
<br/>　　另外，他的武技也有所增长，噬火百刃，如今就是不借助烈焰手套，也能同时发出三十道火刃。
<br/>　　“呼。”萧逸结束修炼，吐出一口浊气，自语道，“还有半个月便是裂天剑派开山收弟子之日，去猎妖殿把任务交了，便不再接任务了，直接去裂天剑派。”
<br/>　　说罢，萧逸站起身，往百武城而去。
<br/>　　百武城，便是这方圆十数城中的最强之城，也有一座猎妖殿坐落此地。
<br/>　　进了猎妖殿，萧逸到任务处出示令牌。
<br/>　　工作人员看了眼令牌，翻查了一下任务手册，做了些记录，便再次将令牌归还给萧逸。
<br/>　　“资料显示，这些山贼团伙都已被剿灭，任务完成。每个任务30点，33个人，总共990点。任务点已记录，这是你的赏金。”
<br/>　　猎妖殿的人，有着自己的规矩，说的话，做的事，几乎都一样。
<br/>　　萧逸接过赏金便准备离开了，但这时，任务处的这工作人员礼貌地问了一声，“易霄先生，还有什么事吗？”
<br/>　　萧逸脸一黑，瞪了这工作人员一眼，暗道，“这么礼貌做什么，你不说话会死阿。”
<br/>　　他已经预料到会发生什么了。
<br/>　　果然，周围无论是交任务的，还是准备接任务的猎妖师，目光全都聚集到他身上。
<br/>　　“易霄？”
<br/>　　“最近忽然声名大噪的那个天才猎妖师？”
<br/>　　“一身黑袍，脸戴面具，少年模样，果然是易霄。”
<br/>　　以萧逸这般疯狂的速度，几乎两天就清剿一个大型山贼团，不出名才怪。
<br/>　　只是，他的名气只限于猎妖师之间。
<br/>　　所以，他从走入百武城，一直到进入猎妖殿，都没有人注意他。
<br/>　　毕竟，戴着面具行走的武者多的是。
<br/>　　没有人会多看他一眼。
<br/>　　哪怕是进入了猎妖殿，周围的猎妖师也不会留意他。
<br/>　　这样衣着打扮和外表的猎妖师，同样多的是。
<br/>　　没有哪个猎妖师会多看他一眼。
<br/>　　但，那猎妖殿的人员说出一声‘易霄先生’，这就引起猎妖师们的注意了。
<br/>　　猎妖师们再打量了萧逸几眼，便认出了易霄。
<br/>　　萧逸无奈地摇摇头，快步离开了猎妖殿，去了下一个城池。
<br/>　　这里距离裂天剑派已经很近了，他不用特地为了完成任务而耽搁时间的话，全速赶路，四、五天就能去到。
<br/>　　这时，他反倒不急了，路过百武城周围的其他城池，也可以慢慢欣赏一下异域风情。
<br/>　　这里，可是距离紫云城有四十多城的距离了，远得很。
<br/>　　对比破玄城、百武城这些有猎妖殿坐落的大城，周围城池的猎妖师少很多。
<br/>　　猎妖师大多会在拥有猎妖殿的城池活动，方便接任务和出任务。
<br/>　　其他城池的猎妖师则只有零星一些。
<br/>　　萧逸随意找了个茶馆坐下，打算歇息一番再赶路。
<br/>　　忽然，附近桌位两位客人的谈话却吸引了他的注意。
<br/>　　看那两人的装扮，似乎是猎妖师，要么就是在外行走的武者。
<br/>　　“诶，你听说了吗，咱们猎妖师内出了很了不起的猎妖师。年龄不大，却实力强悍，连许多老牌猎妖师都不如他呢。”
<br/>　　“我知道，那个易霄嘛。听说只是个少年，还是个控火高手，将那些大型山贼团伙烧得屁滚尿流。”
<br/>　　“可惜阿，这个忽然崛起的天才武者，可能要有大麻烦了。”
<br/>　　“哦？”
<br/>　　“你不知道吗？他一路横扫，都灭了数十个大型山贼团了。现在其它城池的山贼个个都恨他入骨，恨不得将他挫骨扬灰呢，特地出高价请了暗影楼的杀手，誓要杀掉他。”
<br/>　　“暗影楼？真是该死，那些山贼团伙本来就无恶不作，死有余辜。现在竟然请动了暗影楼，那易霄怕真的有大麻烦了。”
<br/>　　两人聊着，都同时叹了几口气。
<br/>　　忽然，他们看到另一边的萧逸，顿时噤声。
<br/>　　“诶，你看，那不会就是天才猎妖师，易霄吧。”
<br/>　　两人都看了眼萧逸。
<br/>　　下一秒，另一人摆摆手，笑道，“开什么玩笑，你以为戴个面具就是易霄了。按照传言，那个天才现在可能在某个山贼窝里大开杀戒吧，怎么可能会来这里。”
<br/>　　两人继续畅聊着。
<br/>　　.....
<br/>　　“暗影楼。”萧逸暗暗皱眉。
<br/>　　他听说过暗影楼，乃是北山郡最庞大、最有名的杀手组织。
<br/>　　同时也是北山郡的大势力之一，比之北山慕容家还要强，势力还要雄厚。
<br/>　　楼中杀手来无影去无踪，没人知道他们的总部在哪里，也没人能找到他们的杀手。
<br/>　　只知道，当有人需要请动他们时，他们会自动出现。
<br/>　　整个北山郡，怕是只有裂天剑派能压他们一头。
<br/>　　“免得夜长梦多，还是先赶去裂天剑派吧。”萧逸想罢，起身再次赶路。
<br/>　　.....
<br/>　　城外，萧逸一路疾驰，然而，还未走几里路，便猛地停了下来。
<br/>　　他明显感觉到，有数道气息锁定了他。
<br/>　　“呵。”萧逸淡淡一笑，“两位，自我出城门，你们就一直尾随，到底是何意？”
<br/>　　萧逸周围空无一人，他仿佛在对着空气胡言乱语。
<br/>　　“哼，莫非要我亲自请你们出来吗？”萧逸冷哼一声，对着身后打出两道火刃。
<br/>　　嗖嗖，两道人影瞬间出现。
<br/>　　“小子，有些本事，不愧是天才猎妖师易霄，警觉性挺高的。”
<br/>　　出现的两人，一个一身黑衣，一个一身白衣，尽皆戴着一个面具，面具上有一小小的‘影’字。
<br/>　　“呵。”萧逸仍旧只是笑笑，再次对着空气说道，“另外的两位，也要在下请你们出来？”
<br/>　　嗖嗖，又是两道人影现身。
<br/>　　“小子，连我们都被你发现了，倒是厉害。”
<br/>　　这两人，同样戴着面具，看不到样貌。但一个是独臂，一个瘸了条腿。
<br/>　　萧逸脸色一冷，“天残地缺，黑白无常，暗影楼四大金牌杀手都出动了，还真是看得起在下。”
<br/>　　“哦，小子你知道我们？”
<br/>　　“不过，既然我们现身了，想必你也知道我们的目的。各大山贼首领出高价要你的人头。”
<br/>　　黑白无常同时冷声说道。
<br/>　　天残地缺则扔出一把匕首，阴声道，“是你自己动手，还是我们来。”
<br/>　　锵的一声，匕首跌落地面，清脆而冰冷。
<br/>　　“提醒你一句。”黑白无常冷声道，“若是让那两个家伙出手，你会死得很惨。”
<br/>　　“相信我们，你虽然是天才，但小小年纪，不会想承受那种折磨的。”
<br/>　　“呵。”萧逸仍旧只是笑笑，毫无动作。
<br/>　　黑白无常见状，冷声道，“看来你并不打算给自己个痛快。”
<br/>　　天残地缺则阴笑起来，自顾地说道。
<br/>　　“他的两只手臂，归我了。”
<br/>　　“他的两条腿，给我玩。”
<br/>　　黑白无常冷笑一声，“随你们，留下他的头回去交任务即可。”
<br/>　　这时，萧逸终于说话了，“看来四位很自信。”
<br/>　　“你觉得你还有逃命的机会？”黑白无常冷笑一声。
<br/>　　萧逸笑道，“刚才城中茶馆，故意在我旁边说话的，就是你们两位吧。”
<br/>　　“你早就发现了？”黑白无常顿时一惊，“那你还出城？”
<br/>　　不错，刚才在茶馆聊天的，便是黑白无常，为的就是引萧逸出城。
<br/>　　萧逸也早就发现了他们。
<br/>　　“因为，我...手痒了。”萧逸直视四大杀手，淡淡地说着，手中凝聚出一股火焰。
<br/>(本章完)
