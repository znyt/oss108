　　第393章
<br/>　　“殿下，在这里住得可还舒服？”宋伯康立在李行哉面前，脸上陪着笑意。
<br/>　　李行哉躺在罗汉床上，把翘起的二郎腿换另一只脚压着。
<br/>　　“蛮舒服啊，舒服得我什么都忘了。”
<br/>　　李行哉来到凉州城已经有些日子，三日一小宴，五日一大宴，凉州城的当红舞姬他几乎都见了一遍，宋伯康的招待不可谓不殷勤，李行哉过的日子不可谓不舒服。
<br/>　　可说这话时，李行哉心里却咬着牙，只用极好的涵养压抑着怒火。
<br/>　　宋伯康哈哈笑着，似乎没觉察到李行哉的愤怒。
<br/>　　“启禀殿下，一个好消息。”
<br/>　　“喔，我在这里还能听到好消息，真是难以相信。”
<br/>　　“的确是好消息，长安城那里传来的。”
<br/>　　李行哉表情一怔，他自从来到凉州城，就和福德勒被软禁在这院子，外界的消息完全被切断。而长安城方面的战局，始终是他最为关心的。
<br/>　　“喔，什么消息啊？”李行哉抬头看着屋檐上的天空，一朵浮云聚了又散。
<br/>　　“大好消息，贼匪在长安城外被击退，退兵伏虎关，破贼之日可期。”宋伯康兴奋道。
<br/>　　李行哉沉默着，从宋伯康对自己的态度，可以看出长安方面的情况。帝国如果守得住长安，那么挟皇子以令诸侯的计策就没多大用，而为了避免来日自己的报复，宋伯康就有可能杀人灭口。而如果贼匪破了长安，天下必定大乱，自己这皇子的身份，就太重要了。说不得，自己还有可能被推到皇位上。
<br/>　　好消息？坏消息？
<br/>　　现在李行哉也算不清什么是好消息，什么是坏消息了。
<br/>　　当然，现在宋伯康是自己唯一的消息渠道，至于他说得是真是假，李行哉也无法分辨。
<br/>　　远处，福德勒坐在假山上，冷眼看着宋伯康和李行哉磨叽。虽然随着李行哉这么久，他已了解帝国大部分事情，可仍旧读不懂这些虚伪的客套。直来直往，强弱之分，这不是很简单的道理么，为何帝国人却习惯藏着掖着。
<br/>　　“宋大人，既然长安战局有利，那咱们何时出兵。再不去，贼匪都被杀干净了，功劳都被我那些哥哥们抢光了。”
<br/>　　“不瞒殿下，我一直在为出兵做准备，可行军打仗不是小事，一切都得从长计议。”
<br/>　　“需要长议到什么时候啊，该不会等到明年庄稼长出来吧。”
<br/>　　“放心，快了，快了。”
<br/>　　等宋伯康走了，福德勒才凑过来，目光瞥了李行哉一眼。
<br/>　　“怎么我感觉你恢复皇子身份后，过得还不如以前流浪的时候。”
<br/>　　李行哉叹口气：“一入宫门深似海，只恨生在帝王家呐。”
<br/>　　“我们被困在这里几天了？”福德勒。
<br/>　　“七天。”李行哉立刻回答道。
<br/>　　七天时间说长不长，说短不短，放在土里，种子都来不及发芽。可搁在长安，甚至已能决定一场战场的胜败。
<br/>　　“我们当想办法离开这里。”福德勒。
<br/>　　远处阿珠阿翠向这边望着，李行哉笑着冲二人挥挥手，嘴里道：“你觉得我不想走么，可咱们怎么走？”
<br/>　　“联系大当家，告诉他我们现在的情况。”福德勒道：“凉州城当有琴川关的暗线。”
<br/>　　“你说得也有道理，可我们现在连这院子都出不了？”李行哉道：“怎么联系琴川关。”
<br/>　　福德勒沉默片刻，道：“好好想，办法总是能想出来的。”
<br/>　　……
<br/>　　宋伯康踏出了院子，李行哉无法看到院子外的情形，整个院子十步方圆都布上了重兵，在高处还有人监视，李行哉的一举一动都尽收眼底。
<br/>　　虽然整个长安都传说六皇子是个废物，但宋伯康依旧很小心。他本身就是个很谨慎的人，这也是崔相派他来主持凉州的原因。
<br/>　　一路穿过回廊，走入大厅，宋伯康刚刚坐下，长史陆洛就等着向他汇报。
<br/>　　这些日子，关于长安城的情报源源不断送到这里。宋伯康倒是没有对李行哉撒谎，在长安城外，朝廷的确打赢了一仗，正义教退兵伏虎关，可长安城的情况依旧不容乐观。
<br/>　　正义教守着伏虎关，援军攻不进去，而坚持的时间越长，长安城便越危险。
<br/>　　“大人，相府那边传来消息，问我们殿下的情况。”陆洛。
<br/>　　“如实回禀，告诉丞相大人，殿下现在在我们手中，出不了什么岔子。”宋伯康说罢，又问了一声：“丞相大人准备得怎么样了？”
<br/>　　陆洛点点头，眼神中闪过一丝兴奋：“都已准备妥当，长安城破，就是我们起事的机会。江南、豫州、包括凉州，到时候我们奉天讨贼，必能平定天下。”
<br/>　　这天下虽然是李家的，但其下还有不少世家。这些家族传承数百年，根深蒂固，无论谁做龙庭，对于他们的身份其实都无甚影响。
<br/>　　李家气数已尽，这是很多人早就看出来的事情。已经到了改天换日的时候，崔家、陆家、宋家、戴家……这些大家族需要重新找代言人，或者，他们自己做主子。
<br/>　　群雄逐鹿，对于那万万人之上的位置，不可能有人不觊觎。
<br/>　　“长安城破，我们就要立六皇子为帝。”陆洛道：“丞相大人的意思是，六皇子好控制么？”
<br/>　　“在京城他就有废物皇子的称号，明帝那么多子嗣，他是最没有可能继承皇位的一个。”宋伯康道：“可毕竟他失踪了很长时间，再次出现却在程大雷手中。谁知道他有没有什么变化。”
<br/>　　“可从这些日子判断，他也不过是个花天酒地的酒囊饭袋。”陆洛。
<br/>　　宋伯康点点头：“你说得也有道理，不过他第一天就想离开，这种警惕还是不能小觑。”
<br/>　　“大人，他不过是个幌子，等平定了天下，随便找个机会除掉他，他在我们手中还能折腾起什么波浪。”陆洛。
<br/>　　宋伯康沉默片刻，仔细思来，李行哉却是已经被完全控制，倒是不必对他过分上心。目前他关心的，当然也是整个天下关心的，还是长安城的战况。
<br/>　　“你说，有没有可能，李家守得住长安城？”宋伯康不无担心道。
<br/>
