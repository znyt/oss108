　　第527章
<br/>　　“嗯。”刘发财下意识答应一声，忽然激灵一下，睁大眼睛看着程大雷。
<br/>　　程大雷满脸堆笑：“在下刘发财，一直仰慕程当家为人，愿在大当家手下做事，鞍前马后，在所不辞。”
<br/>　　刘发财眯起了眼睛，上下打量着程大雷。穿一件灰不溜秋青布袍子，一张黑里泛红的丑脸膛，面前立着一头长毛黑牛。
<br/>　　这样一个人，无论怎么看都是平平无奇。
<br/>　　但为何他偏偏也叫『刘发财』。的确，这三个字不算特殊，帝国上层给子孙起名字，自然引经据典，但底层平民，往往是抓到什么算什么，发财、狗剩、栓子，石头之类的，经常会出现。
<br/>　　可是，在此地碰见一个同名同姓的，是不是也有些太巧了。
<br/>　　“嗯呐，这名字不错，有前途。”刘发财打了个哈哈，掩饰掉脸上的诧异，道：“你既然认出我是谁，刚才就是故意出手了？”
<br/>　　“阁下武艺高强，怎么会对付不了几个小蟊贼。即便我不出手，肯定也能化险为夷，转危为安。”
<br/>　　“呃，哈哈哈。”刘发财开心的笑道：“没想到被你看出来了，几个小蟊贼自然不足挂齿，最近我吃素，懒得多造杀孽而已。”
<br/>　　这种事，刘发财其实并非第一次遇见。他打着程大雷的名头招摇撞骗，有些山贼遇见了就奉为上宾，好吃好喝好招待，还有些人纳头便拜，要给刘发财牵马执蹬。
<br/>　　当然，后来被识破，满天下追杀的事也经常出现。
<br/>　　所以刘发财对于程大雷的出手，并不觉得意外。只不过他的名字……
<br/>　　他笑着打量程大雷的黑牛一眼，又拍拍程大雷的肩膀，道：“倒是有膀子力气，看着不错。”
<br/>　　“如果能跟在阁下身边，学个三招两式也够我一辈子受用了。”
<br/>　　“哈哈，不能这么说么。我的武艺也只是泛泛，你也不要太看不起自己。”
<br/>　　程大雷心里点点头，的确，你的武艺最多也就是泛泛，不然，也不会被人赶得像条狗一样。
<br/>　　不过，程大雷对这个刘发财还蛮感兴趣的，拉到山寨多少能发挥些作用。
<br/>　　而刘发财也看中了程大雷一身力气，以后再遇到危险也可以出些力。实在不行，自己也完全可以将他卖了嘛。
<br/>　　“程当家这是要去哪里？”程大雷问。
<br/>　　“难道你没有听说么，兖州城建了一座招贤馆，招募天下英雄，四方高手都想凑这次热闹，我也过去看看。”
<br/>　　“咦，程当家难道要投兖州？”
<br/>　　“呃……”刘发财一楞，差点露馅了。他哈哈笑了一阵，道：“我用得着投靠谁，无非是去凑凑热闹。你知道么，我这个人最是喜欢凑热闹。”
<br/>　　“喔……”程大雷道：“其实我也蛮喜欢凑热闹的。”
<br/>　　见程大雷没有发觉，一副傻乎乎的模样，刘发财松了口气，问道：“你是从哪里来的？”
<br/>　　程大雷随口编了个谎，只说自己来自某个叫刘家村的地方，自幼习武，有膀子力气，活到这个年纪，想要在江湖上闯荡一番，拼一个前程。
<br/>　　对于他的话，刘发财也没有觉得奇怪。现在帝国多的是这班人，不管有本事还是没本事的，都自称有本事，想要投到各路诸侯王门下，混一个前程。习得文武艺，货与帝王家嘛。
<br/>　　二人你一句我一句，互相嘴里没有一句实话，可是却结伴而行。一路沿着黑河走，走到原先石桥处，见没人再守着石桥，二人麻溜快的过桥进入了兖州。
<br/>　　过了桥，刘发财竟找到了那头他丢失的黑牛。他心情顿时畅快起来，毕竟，这年头一头牛可不便宜，尤其还是黑色的。至于丢掉的斧头，估计是被独角鬼捡走了，回头自己再找好铁打一柄就是。
<br/>　　“那个……黑河九鬼是什么来头？”程大雷问。
<br/>　　“他们的名号我倒是听说过，黑河九鬼一共九人，是结拜兄弟。老大独角鬼，就是今天咱们遇见那人，他们占了上游的鹰嘴岭，寨里有三五百弟兄。”
<br/>　　“啧……”程大雷叹了一声：“咱们不会遇到他们吧。”
<br/>　　“哈，我怕还遇不见他们呢。如果他们不露面，我就放他们一条生路。如果他们还敢露面，那就莫怪本大家大开杀戒。”刘发财骑在黑牛背上，心里想的却是：该不会这么倒霉吧。
<br/>　　两个人骑着两头黑牛，并肩而行，看身高模样，竟差不了多少。
<br/>　　正此时，前方传来一阵喊杀之声。黑天半夜，远处亮着火把。
<br/>　　“前面好像打起来了。”两人同时道。
<br/>　　“咱们可别过去！”
<br/>　　二人又是异口同声，说罢之后，互相看了一眼彼此。都有一种：英雄惜英雄，英雄重英雄的感觉。
<br/>　　可是，二人不想多管闲事，麻烦却会主动找上来。
<br/>　　前方两队人撞在一起，一队人杀，一队人逃。逃着逃着竟向这边逃了过来。
<br/>　　程大雷第一眼看见的，就是那年轻商人。他身后是大波的山贼，大概有一二百人。
<br/>　　刘发财已经变了脸色，想着三十六计，走为上计。可是，这批山贼的头领正是那独角鬼。
<br/>　　此刻仇人相见，分外眼红。他拍马赶了过来，口中大吼：“与我杀了他们。”
<br/>　　刘发财想走也来不及，他座下黑牛的脚力他是知道的，也就是装装样子，肯定比不过对方的战马。
<br/>　　只不过他虽慌不乱，高昂头哈哈笑着：“来，本当家等你们多时了。那个不想活的，来来来……与我兄弟说话。”
<br/>　　程大雷正听得认真，忽然感觉肩膀上被重重拍了一下。回过头，只见刘发财看着自己认真道：“兄弟，靠你了！”
<br/>　　“咦！”
<br/>　　程大雷觉得，在无耻这件事上，刘发财是完全可以和自己匹敌的。
<br/>　　独角鬼已经带着人杀到，刘发财掉过牛头便逃，倒是将程大雷晾在那里。
<br/>　　独角鬼握着铁棍，拍马追向刘发财，对于程大雷，他倒是不如何看重。
<br/>　　但想要追上刘发财，总是要从程大雷身边经过，二人相错身的刹那，无人知道发生了什么，只见独角鬼跑着跑着……
<br/>　　人头忽然掉在地上。
<br/>(本章完)
