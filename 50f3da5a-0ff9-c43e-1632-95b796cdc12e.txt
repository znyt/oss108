　　第413章
<br/>　　程大雷一颗心揪了起来，兵力比达到如此可怖的程度，一旦开战，定然是巨大的牺牲。
<br/>　　赵子龙的瞳孔却有些炙热，座下战马也显得躁动不安，恍惚准备冲出去厮杀一阵。
<br/>　　“大当家，龙字军全员待命，只等大当家一声令下。”
<br/>　　程大雷回过头，望着如山头般林立的龙字军，从每个人炙热的瞳孔中，程大雷感觉到了某种叫战意的东西。
<br/>　　他忽然心中一动，心道：是呐，龙字军一直拥有最好的装备，最好的给养。这种『最好』，不仅是在山寨，放眼整个帝国也同样数一数二的。
<br/>　　他们一直接受最残酷的训练，与山寨其他队伍也经常切磋，甚至有时也会出现伤亡。但演义并不是真的打仗，真实战斗的残酷性，绝不是靠想象就可以体会的。
<br/>　　他们究竟能不能打仗，上到战场是奋勇杀敌，还是会吓得尿裤子。
<br/>　　不如，让他们试试。
<br/>　　程大雷蠢蠢欲动，不知不觉间，摁剑的手也变得炙热。
<br/>　　程大雷刚才还批评赵子龙喜欢逞匹夫之勇，然而他自己，又何尝不是呢。
<br/>　　“接下来由你指挥，随机应变，如果真的需要……”程大雷顿了顿：“你便大开杀戒。”
<br/>　　赵子龙眼睛一亮，立刻道：“是！”
<br/>　　“龙字军，准备！”赵子龙举起长枪。
<br/>　　“是！”三千人同时高呼回应。
<br/>　　“出阵！”
<br/>　　“威！”
<br/>　　三千人向前移动，走出山坳，坐下马安静不发出任何声音。龙字旗此刻已高高扬起，迎风招展。
<br/>　　程大雷满意的看着这一幕，他并不想越俎代庖，抢赵子龙的指挥权。这支队伍由他亲自调教，亲自训练，赵子龙自然也是最好的指挥者。
<br/>　　对于赵子龙的战斗力，程大雷不用怀疑，也希望他的战斗力同样出色。接下来，就是拭目以待的时刻。
<br/>　　三千人的队伍矗立在荒原上，人如虎，马如龙，龙马精神。
<br/>　　程大雷在队伍最前方，目视着远处的黑夜，隐隐已听到远处的马蹄声。
<br/>　　人马来得比程大雷想象中更多，从发现李行哉后，凉州兵便向城西集结。因为宋伯康答应对救回李行哉的人将由重赏，对于这种升官发财的事情大家自然谁也不会落后。所以士兵争先恐后往西边追来。
<br/>　　如此，便没有了阵形，凉州兵不是成方阵，而是成一条线。
<br/>　　李行哉没命向前狂暴，对于这种事他早已习以为常。因为在过往流浪的日子中，他被人追得像条狗的日子实在不计其数。
<br/>　　但从没有一次，像今天一样迫切的不想被抓住。以前无非就是偷只鸡，偷只狗，被抓到也无非就是被揍一顿，然而今天，身后这帮人想要的是帝国的江山。
<br/>　　李行哉不想当傀儡。
<br/>　　宋伯康竟然在追兵队伍的最前方，手下人不由惊讶，一向以儒生面目示人的州牧大人，竟然有如此出色的骑术。
<br/>　　凉州兵追得越来越急，双方之间的距离也在不停缩短，三百步，二百步……李行哉心急如焚，忽然见到前方出现一支兵马，长矛如林般耸立，程大雷骑着黑牛，背着大斧出现在队伍最前方。
<br/>　　李行哉从未像今天一样，觉得程大雷这张丑脸竟如此亲切。
<br/>　　“程大雷，我来了……”
<br/>　　忽然，从身后射过来一箭，李行哉身下马重重倒地，李行哉也跌了出去。
<br/>　　李行哉看到程大雷时，宋伯康同样也看到了对方。如果让李行哉落到程大雷手中，再想救过来对方便不容易了。
<br/>　　追击过程中，宋伯康一直没下令射箭，到最后他还是希望活着抢回李行哉。此时选择射箭，也是无奈之举。
<br/>　　首先射伤了对方的马，让李行哉坠马。同时王德和白元飞已经奔出一段距离，眼见凉州兵已经追过来，再想回去抢人已经来不及。
<br/>　　“放箭！”
<br/>　　赵子龙的手重重挥下。
<br/>　　然后箭雨如织，生生截停了凉州兵的冲击。宋伯康挥手让手下人停止进攻，与此同时，赵子龙也暂时摁住要冲锋的队伍，看宋伯康想做什么。
<br/>　　双方隔着二百步距离对峙，东方蒙蒙有了亮光，这样距离已能清晰得看见彼此的脸。而李行哉，恰恰就在两支队伍中间。
<br/>　　他从疾奔的马上跌下来，此刻摔得不轻，幸运的是都是擦伤和淤青，并没有丧失行动力。
<br/>　　宋伯康在队伍最前方大叫：“殿下，我是来救您回去的，您快快和我一起回去吧。”
<br/>　　李行哉从地上坐起身，疼痛令他嘶地倒抽一口凉气，随之脸上挂起笑容。
<br/>　　“宋伯康，我劝你还是死了这份心吧，现在本殿下已经活着出来了，哈哈，谁也别想再抓住我！”
<br/>　　哎呦，李行哉忽然咧起嘴，刚才笑得太猖狂了，不小心扯动伤口。
<br/>　　“臣乃是一片耿耿忠心，日月可鉴，殿下莫要误会臣。我当真是是来救殿下的……”
<br/>　　“救？呵呵。”李行哉发出一声冷笑，道：“怎么，还要抓我回去当你们的傀儡，姓宋的，你小看本殿下了，等日后本殿下翻身，第一个就砍你人头。”
<br/>　　宋伯康深吸一口气，道：“殿下……”
<br/>　　“呔，大胆逆贼，休要再鼓唇弄舌。”李行哉从地上站了起来，虽然衣服破烂不堪，还沾着污垢，但脊背却无比挺直：“本殿下是要去长安破贼的，等破了长安之贼，再来诛你这帝国逆贼。”
<br/>　　有件事一直悬在李行哉心底，便是长安之围。身为人子，身为帝国皇子，他都毫无疑问该站在风浪之巅，而不是躲在凉州，被人软禁。所以过去的每一天，对于他来说都是度日如年，可想而知他此刻是多么愤怒。如果有可能，就是将宋伯康大卸八块，生吞活剥也难消李行哉心头之恨。
<br/>　　宋伯康刚才的话被打断，此刻看着李行哉，犹豫了片刻。
<br/>　　“启禀殿下，有件事我也是刚刚得到消息，正要禀报殿下，没想到殿下便被贼人劫走。”宋伯康顿了顿。
<br/>　　“呵，我们已经撕破脸了，你还想说什么？”
<br/>　　宋伯康抬起头：“长安已破，就在七日前。”
<br/>
