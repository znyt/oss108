<br/>    魔宗六劫继续朝着周围追去，不过一个时辰之后。
<br/>    
<br/>        魔宗六劫，排行第六的魔宗排名第七的弟子看向身前，一条偌大的剑龙正在十多位魔王之间纵横，“陈明休走，看我来战你！”
<br/>    
<br/>        排行第七的魔宗弟子冲到了陈明的面前，口中暴喝道：“魔道天功！”
<br/>    
<br/>        他的身体急剧膨胀，浑身肌肉纠结，青筋鼓起，如同一条条小蛇在身体之中窜动。
<br/>    
<br/>        陈明透过剑龙，有些好奇地看向地面上的那一把魔刀，嗯，现在追到了我的这个小家伙没注意到吗？
<br/>    
<br/>        魔宗弟子也是看向地面之上，还有着一柄魔刀在水中缓缓下沉，卧槽！
<br/>    
<br/>        我来到现场的时候，老三就只剩下一把刀了！我现在正在正面面对陈明大魔王，怎么办，在线等，急！
<br/>    
<br/>        陈明笑呵呵地看向身前的魔宗弟子，笑道：“话说你们魔宗还真是扣，堂堂魔宗七劫，连一把神兵都没有，搞得我完全没有兴趣杀你们啊！”
<br/>    
<br/>        魔宗弟子大喝了一声，“陈明，休得胡言，看我斩了你！”
<br/>    
<br/>        魔宗弟子道完，浑身法力爆涌，瞬间已经是后退了十里，消失在了黑暗之中。
<br/>    
<br/>        陈明愣愣地看着，“还真是有我年轻时候的风范！而且，果然不是主角的待遇，如果是主角的待遇，对方明明知道不是我的对手，知道我可以越阶斩杀，还是会过来送死的，不得不说凌仙的光环是神器啊！”
<br/>    
<br/>        卓青瑶疑惑道：“师父，光环是什么？”
<br/>    
<br/>        陈明道：“就是说你二师弟很笨，但是，其他人比他还要笨！”
<br/>    
<br/>        卓青瑶思考了片刻，然后道：“二师弟其实也没那么笨，也就一般笨而已，嗯，比猪还是要聪明一点的！”
<br/>    
<br/>        魔宗弟子一边走一边骂骂咧咧，“到底是哪个混蛋，想出来直接上去跟他打的，这不是找死吗？”
<br/>    
<br/>        这时，魔宗七劫的老大正从老七的面前经过，老大问道：“老七，你可曾找到了陈明！”
<br/>    
<br/>        老七愤愤道：“找到了。”
<br/>    
<br/>        老大喝问道：“那你为什么没有缠住他！”
<br/>    
<br/>        老七道：“我找到陈明的时候，老三已经只剩下一把刀了。”
<br/>    
<br/>        老大：.......
<br/>    
<br/>        “算了，我们还是一起去追杀他吧，不过阵法师也太难杀了吧，布阵根本没用，他想跑几乎没有任何办法。”
<br/>    
<br/>        陈明看了自己的进阶任务一眼，还差五十三人。
<br/>    
<br/>        再杀五十三人就好了，自己就可以完成自己的进阶任务了，五十三人，找上一个王级军团，很快就可以搞定了。
<br/>    
<br/>        陈明从黑暗之中消失，向着更远的地方前去，出口处，他没有任何兴趣，被大君逮住，加上十多个王级军团，就真的是死无葬身之地了。
<br/>    
<br/>        搜寻了三天，陈明看得到，黑暗的水中，一具具人王的尸体正在缓缓下沉，似乎是雪山宗和群星仙门的，陈明循着尸体一路寻找了过去，前方正有着两支王级军团，正在追杀紫霞仙子和雪自悟，紫霞仙子现在的状态也是不太好，嘴角有着鲜血溢出，一只胳膊已经软踏踏的，似乎断了，雪自悟更是浑身是伤。
<br/>    
<br/>        他们虽然是悟道榜上的天才，只是遇到两个王级军团，还有的一战，但是这两个王级军团之中，还有着魔宗七劫剩下的五人。
<br/>    
<br/>        陈明挥手之间，万道天竹抚动，两千四百柄灵剑出鞘，竹叶飘飞，瞬间化作一柄柄灵剑，朝着众多的魔王飞去，磅礴的剑意四射，紫霞仙子和雪自悟回头，紫霞仙子的眼中露出一抹惊喜之色，雪自悟更是大喊道：“天无绝人之路，是宗师！”
<br/>    
<br/>        剑龙席卷之下，十多位魔王直接被搅成了碎片，一阵血花飞舞，将周围彻底染成了血红色。
<br/>    
<br/>        魔宗七劫剩下的五人大惊失色，连忙道：“速速结阵！”
<br/>    
<br/>        剩下的黑衣人瞬间结成阵型，围成一个圆球，陈明目光所及，全是魔气森森的长刀。
<br/>    
<br/>        陈明扫了一眼，笑道：“还真是粗劣的伎俩！”
<br/>    
<br/>        剑龙飞舞，直接朝着对方凝聚的圆球冲去，无数魔刀飞射，朝着剑龙劈去，一簇簇火花迸溅，剑龙飞行在魔刀之中，如同在冒着刀雨前进，剑龙浑身的灵剑旋转，瞬间将自己组成了一个漩涡，陈明本想直接控制剑龙冲进王级军团之中屠杀，却是被王级军团之外的魔气弹开。
<br/>    
<br/>        陈明扫了一眼，笑道：“有点意思啊！”
<br/>    
<br/>        有点像血衣剑卫的方式，不过没有血衣剑卫那么强，魔气可以交融其中吗？
<br/>    
<br/>        剑龙矗立在王级军团之前，看向对面六十多位魔王，剑龙的口中，忽然吐出一柄魔刀。
<br/>    
<br/>        魔宗七劫剩下的五人看向这一柄魔刀，“那是老三的魔刀！”
<br/>    
<br/>        “他把老三的魔刀吐出来干嘛？”
<br/>    
<br/>        “难不成，他想把老三的魔刀还给我们不成？”
<br/>    
<br/>        也不怪他们，任谁看到一个在剑道之上数次人剑合一的家伙，吐出一柄魔刀来都有些疑惑。
<br/>    
<br/>        然后他们就看到，陈明站在剑龙的头顶之上，脚踏剑龙，直接朝着魔刀追去，陈明花了一千点师德，直接达到血海魔刀人刀合一的地步，骤然间，魔气荡漾！
<br/>    
<br/>        然后所有人就看到，一个施展了数次人剑合一，一个修行剑道跑偏了的家伙，施展了一次人刀合一，“血海无归！”
<br/>    
<br/>        一刀之下，魔气彭拜，鲜红色血雾如同阴云一般笼罩数十丈，锋锐的刀光横掠，刀光随意飞入魔气之中，没有受到任何阻碍，直接将三位魔王斩杀！
<br/>    
<br/>        魔宗七劫之首道：“刚刚，那是人刀合一？”
<br/>    
<br/>        又有一位魔宗七劫道：“不仅仅是人刀合一，还有着魔气，他修行了魔功！”
<br/>    
<br/>        不仅他们疑惑不解，就是紫霞仙子和雪自悟都愣住了，雪自悟道：“我一直以为，宗师只是在剑道上跑偏了，看来我错了，他在战斗方式上，已经是彻底跑偏了！”
<br/>    
<br/>        紫霞仙子道：“不仅仅是战斗方式，他还修了魔功，他在修行上，也是彻底跑偏了，不过，宗师好像并没有入魔的感觉啊！”
<br/>    
<br/>        这一点雪自悟倒是可以解释，“有些人，都是仙魔同修的，互相借鉴，不过需要庞大的心念和坚韧的道心，宗师估摸着是布阵和炼丹需要，就去修了魔功。”
