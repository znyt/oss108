        &lt;h3&gt;第三卷第二百八十一章  偶遇魔女彭慧&lt;/h3&gt;
<br/>
<br/>        陈子昂带着易倾秋和谭九儿见了乔小东之后，她又独自去见了一个人。
<br/>
<br/>        刘翼飞现在已经是西城数一数二的人物了，他现在主管西城的开发，和陈子昂的信息化完全不搭边，但是和江俞轩的事业却有着千丝万缕的联系。
<br/>
<br/>        陈子昂刚到西城的时候就是打着陆家的名义见了刘翼飞，然后拿到了西城的第一个蛋糕，而且还得到了李凌菲，这么几年，陈子昂并没有和刘翼飞有过多接触，最重要的原因刘翼飞是陆家的人，当时的她不想过多接触陆家的人。
<br/>
<br/>        陈子昂给刘翼飞打电话，刘翼飞在开会，西城里里外外整个规划都在他的手中，这次会议主要是高新建设和西岭的别墅群。
<br/>
<br/>        这两个项目都是大项目，高新建设将会和西城新区交相辉映，是西城的一大亮点，一个是古文化及现代相结合，一个是完完全全的现代化都市建设，这将是西城发展史上浓墨重彩。
<br/>
<br/>        西岭的别墅群，将会在西岭群山里或者是山脚下建立一些高档独栋的别墅，这个项目方案也已经落地，项目也将进入实施期，别墅还没有建好，已经被预定了很多套。
<br/>
<br/>        高新区的规划方案也已经落地，在众多的房产商里，江氏集团脱颖而出，原本上面是会否决江家方案的，所以刘翼飞想再开一次会进行一次研究，他其实是看好江家的，不过有些事情他得权衡利弊。
<br/>
<br/>        刘翼飞看了一眼电话，“陈子昂。”他在脑海里搜索了一下，记起来了第一次陈子昂兄妹去找他的情景，后来好像都不曾联系过，这让他有点好奇。
<br/>
<br/>        “陈子昂？”刘翼飞轻声的问。
<br/>
<br/>        “刘总，您好，好久不见，您还记得我啊？”陈子昂甜蜜的语气夹杂着欢快，让刘子翼听得心里也莫名的愉悦起来。
<br/>
<br/>        “当然记得啊，你们兄妹郎才女貌，现在的生意做得红红火火，不由得让人要记住啊。”刘翼飞轻笑着。
<br/>
<br/>        “谢谢刘总。有没有打扰到您？”
<br/>
<br/>        “我正在开会。”
<br/>
<br/>        “那就不打扰您了，我回头在联系您，刘总再见！”陈子昂干脆利索的挂了电话。
<br/>
<br/>        刘翼飞挂了电话接着开会，彭慧冷冰冰的盯着刘翼飞，等着刘翼飞将电话打完：“我们西城乃至西省是要向前发展的，我们需要合格的合理的适合现代化发展的方案，而不是根据某个人的嗜好选择放案，另外，西岭的项目就是一个鸡肋，那是会破坏整个生态环境的。”
<br/>
<br/>        彭慧外号魔女，是西城里最特别的存在，敢说敢言也敢做，她带着眼镜，短发，小巧的鼻翼因为激动而微微抖动着，有些人私心太重，枉顾王法，唯利是图，不顾一个城市的合理发展，强行插入确实让人心疼，她人微言轻，就算说了也起不到太大的作用。
<br/>
<br/>        “你说的我不是没有考虑过，但是.......”刘翼飞
<br/>
<br/>        “但是什么？你掌管着整个西城的规划，你不知道那个方案利于西城的发展？”彭慧粗暴的打断了刘翼飞的话。
<br/>
<br/>        “我觉得都合适。”刘翼飞心里也不舒服，他就是一个受气包，夹在两头为难。
<br/>
<br/>        “你放屁！”彭慧火起，不复斯文破口而出。
<br/>
<br/>        “你.......”
<br/>
<br/>        眼看俩个人吵起来了，柯清荣站起来说：“这个事情能不能再议？我还有一个电话会议要参加。”
<br/>
<br/>        “那就再议吧！”刘翼飞合上笔记本站起来就走了。
<br/>
<br/>        彭慧也气咻咻的拿起文件夹转身下楼，她心里实在是火大，出了大门看见路边停了一辆车，看也没看的就上车，对司机说：“送我去高新办。”
<br/>
<br/>        陈子昂将车停在门口，想着等半个小时之后给刘翼飞打了电话，今天务必见刘翼飞一趟，要不然后期没有时间了。
<br/>
<br/>        可是上来了一个女的气呼呼的直接命令她去高新办，她微微一愣，脑海里立马出现了彭慧的信息：“好咧。”发动汽车朝高新办驶去。
<br/>
<br/>        彭慧坐在后座闭上眼睛微微的平息了一下情绪，睁眼发现开车的是个女司机：“我怎么从来都没有见过你？”
<br/>
<br/>        “嗯，我第一次来。”陈子昂看着后视镜里面的女人微微一笑。
<br/>
<br/>        “第一次来？你不是我们单位的司机？”
<br/>
<br/>        “是的，我只是去你们单位等人，看你急匆匆的上车，我就送你一程。”
<br/>
<br/>        “哦，那倒是麻烦你了，我也是生气了，不好意思啊。”
<br/>
<br/>        “没事，这到高新也不远，反正我下午也没有事情。”
<br/>
<br/>        “你等谁啊？”彭慧问陈子昂。
<br/>
<br/>        “一个很久没有见的朋友，其实也不算是朋友吧，只是我第一次到西城做生意的时候找过他，这次是想见见他，当面说一句感谢的话。”
<br/>
<br/>        “你做什么生意？”
<br/>
<br/>        “信息化上的。”
<br/>
<br/>        “信息化不是夏颜管着吗？你找夏颜？”彭拜皱着眉头。
<br/>
<br/>        “不是，另外的一个人。”
<br/>
<br/>        彭慧沉默了。每个人都有自己的隐私，她好像问得有点多。
<br/>
<br/>        “你是去高新视察工作呢？”陈子昂打破沉默。
<br/>
<br/>        “是啊，希望高新的项目能达到预期的希望吧！”彭慧叹了口气。
<br/>
<br/>        “你这一番忧国忧民的样子让我想起了我好姐妹的哥哥。”
<br/>
<br/>        “谁呀？”彭慧好奇的问。
<br/>
<br/>        “落白。”陈子昂微微一笑，看着后视镜彭慧吃惊的样子。
<br/>
<br/>        彭慧不再说话，看着窗外的路陷入沉思。
<br/>
<br/>        刘翼飞生气的回到办公室，将笔记本和手机往办公桌上一摔，气呼呼的坐在椅子上，这个彭慧就是死脑筋，不就是过一套方案，至于这么带着感情的和他吵吗？
<br/>
<br/>        气归气，想起了刚刚接了陈子昂的电话，陈子昂当时是打着陆家的名义来的虽然之后没有联系，但是他也有所关注，生意做得不错，女的一直在北京，男的在西城，陆家也不曾提起过他们是什么关系，有些事情人家不说他就不问。
<br/>
<br/>        他拿起电话给陈子昂回了个电话：“陈子昂。”
<br/>
<br/>        “您好。”陈子昂并没有喊出刘翼飞的姓，因为彭慧还在她的车上。
<br/>
<br/>        “你找我有事情吗？”
<br/>
<br/>        “嗯，我到西城了，想见一下您，感谢一下这几年您对我们的关照。”陈子昂笑着说。
<br/>
<br/>        “感谢就不必了，下午你有时间来我办公室吧。”
<br/>
<br/>        “好咧。我现在在高新，等会就过去找您。”
<br/>
<br/>        到了高新办，这个是因为高新规划成立的部门，杨东负责。
<br/>
<br/>        “你叫什么名字？”彭慧问陈子昂，免费给她当了一次司机，还不知道人家姓什么。
<br/>
<br/>        “陈子昂！”
<br/>
<br/>        “陈子昂？好特别的名字。我叫彭慧。”彭慧下车，陈子昂也下车，俩个人这才彼此打量起来，都相视一笑。
<br/>
<br/>        “彭总，你怎么来了？咦，子昂，你怎么会和彭总在一起？”杨东准备出门，发现俩个女人面对面对着车傻笑。
<br/>
<br/>        “我来看看一心为公的杨总，子昂送我过来。”彭慧自然而然的接过杨东的话。
<br/>
<br/>        “你们俩认识？”杨东好奇的问。
<br/>
<br/>        “我们俩刚刚认识的。”陈子昂也笑着说。
<br/>
<br/>        “上去坐坐？”杨东看着陈子昂说。
<br/>
<br/>        “我不去了，我约了人一会见面，等晚上我去你家蹭吃蹭喝。”陈子昂调皮的一笑。
<br/>
<br/>        “那你给你姐姐打电话了没有？”杨东想着陈子昂回来可能没有联系他的老婆。
<br/>
<br/>        “这几天特别忙。”
<br/>
<br/>        “行了，你确定要去我一会给她打电话。晚上等你。”
<br/>
<br/>        “好的，晚上见，那我先走了。”陈子昂冲杨东和彭慧挥挥手，开着车一溜烟的跑了。
