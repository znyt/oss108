        &lt;h3&gt;第三卷第一百七十九章  寒家年会&lt;/h3&gt;
<br/>
<br/>        这个新年，安男希望寒伯安能和她一起度过，寒伯安公司的年会是带着她一起出席的，他们是实至名归的夫妻，虽然她很少在寒伯安的公司露面。
<br/>
<br/>        寒家大少的婚姻也并非是一帆风顺，为了不让安男过多的被关注，这几年俩个人都没有公开他们的恋情。
<br/>
<br/>        他的身份和地位在哪里放着，被多少人关注和爱慕，但是寒伯安都不为所动，他和他弟弟一样，是那种对待感情都特别认真的人，不爱就不爱，爱了就往死里爱，溺水三千只取一瓢饮，好的是他比弟弟幸运，爱上了安男。
<br/>
<br/>        当寒伯安带着安男到达年会现场的时候，大家都惊呆了，除了李少卿见过安男外基本没有人见过安男。
<br/>
<br/>        安男挽着寒伯安的胳膊，她穿着寒伯安为她量身定制晚会礼服，身材恰到好处却又不失温婉大方，俩个人款款而来，当他们一出现时，整个晚会安静的能听到彼此的呼吸声，人们的眼光有欣赏有羡慕、嫉妒还有不甘。
<br/>
<br/>        简单的介绍过后，便是筹光交错的应酬，他们的年会不仅仅只是寒伯安公司的内部员工，一些大的家族都会将年会当做彼此联系感情的纽带。
<br/>
<br/>        安男随着寒伯安一起来来去去的应酬着，中间她离开了寒伯安去了一趟洗手间，在洗手的时候，旁边的一个美女对她说：“寒伯安给了你多少钱让你当他的女伴？”
<br/>
<br/>        安男低着头专心致志的洗着手。
<br/>
<br/>        “我说你聋了吗？寒伯安给了你多少钱让你当他的女伴？”语气里满是不屑。
<br/>
<br/>        “谁给了你资格和胆量和我这样说话。”安男抬起头，旁边的女侍递给安男一块擦手纸，安男慢条斯理的擦着手，冷冷的看着眼前气焰嚣张的染着金色头发的女人。
<br/>
<br/>        “你知道我是谁吗？”
<br/>
<br/>        这个女人可以用明眸皓齿来形容，只是那双眼睛里满都是嫉妒和仇恨，身材不错，气质也不错。
<br/>
<br/>        “她是谁？”安男问旁边的女侍。
<br/>
<br/>        “她是寒少这几年的女伴，每次出席会议的时候都带着她，也是寒家旗下的艺人，寒少这几年独宠她一个人。”
<br/>
<br/>        安男：“寒伯安还有女伴？”
<br/>
<br/>        “像他们这种身份的人有女伴不是很正常吗？况且寒少又没有结婚。”女侍像是看怪物一样看着安男。
<br/>
<br/>        “呵呵呵，好像你很单纯一样，确实长着一张单纯的脸，不过不要以为寒少带着你出席一次年会就得意忘形，寒少永远都是我的。”金发美女面带讥笑。
<br/>
<br/>        “你当了他几年女伴都没有搞定他，有何脸面说他是你的？有本事当着他的面说给我听吗？”安男不屑一顾，这种女人啊仗着自己有几分姿色吃着青春饭，老了都不知道有多么凄惨。
<br/>
<br/>        安男觉得和这种女人说话简直就是侮辱了自己的智商，她微微蹙了一下眉头，转身就走，可是那个女人就是不让她走。
<br/>
<br/>        “麻烦你让我过去。”安男的语气已经冷了。
<br/>
<br/>        可是她还是不让，安男有些生气了，用手将她一拨，那个女人身体一歪倒在了地上。
<br/>
<br/>        “寒少，我.....我什么都没有作，她，她动手推我。”金发美女哭唧唧。
<br/>
<br/>        “你怎么在这里？”安男看着走过来的寒伯安。
<br/>
<br/>        “你半天不出来，我来找找你。”寒伯安连看都不看地上的人。
<br/>
<br/>        “你不扶一把她吗？她可是你的女伴。”
<br/>
<br/>        “有时候要出席酒会，没有办法就只能随便找个人陪着了，当然也是为了解决掉一些烂桃花。”寒伯安看着安男那含笑的眼。
<br/>
<br/>        “呵，合着你的烂桃花还很多。”
<br/>
<br/>        “你老公我这样一个人见人爱，花见花开的人怎么可能没有几朵乱桃花来为我招蜂引蝶呢。”
<br/>
<br/>        “哼。”安男轻哼一声。
<br/>
<br/>        “当然，他们只馋我的钱，不馋我的身子。”寒伯安毫不顾忌地上不起来的人，轻轻的拉着安男的手转身离开。
<br/>
<br/>        “记住，今天晚上起，不要再让我看见你，也永远退出娱乐圈，如果让我见到，见一次收拾一次。”寒伯安的声音想寒冷的冰。
<br/>
<br/>        “你是不是太狠了点？”安男有些不忍心。
<br/>
<br/>        “主要是她太没有眼色了，想动不该动的人。还有，你要记住，你家老公在你的眼中不是宝贝，在别人眼中可是宝贝呢。”
<br/>
<br/>        “噗嗤，还宝贝呢？不就是爱招惹烂桃花嘛。”安男忍不住笑了。
<br/>
<br/>        “男男，我是寒家的人，后面你会面临很多的事情，像今天这样的事情可能就是是小儿科，以后我不在你身边的时候切记要保护好自己。”寒伯安正色的对着安男说。
<br/>
<br/>        “我知道了。”
<br/>
<br/>        “另外，和陈子昂的关系也不能走的太近，更不能让别人知道你们有血缘关系。”
<br/>
<br/>        “为什么？”安男不太明白。
<br/>
<br/>        “陈子昂和张函之间的关系错综复杂，张函本身很复杂，他们原本相爱，只是被家里逼着分手了，现在陈子昂带着孩子生活还算是平静，但是如果孩子的事情出现差错，怕是会引发一系列的事情。”寒伯安的语气里有一丝丝忧虑。
<br/>
<br/>        “你这是要放弃子昂吗？”安男的语气突然就冷了下来。
<br/>
<br/>        “不，作为朋友我都不会放弃她的，何况现在她和你还有血缘关系。”寒伯安知道安男误会了他。
<br/>
<br/>        “伯安，我只是希望大家都平平安安的，如果真要选择的话.......”
<br/>
<br/>        寒伯安微笑的看着安男。
<br/>
<br/>        “虽然陈子昂和我有血缘关系，你却是我一生要相守的人。”安男抚上寒伯安的脸颊：“为君生，生而相守，守而相恋、恋而相知、知而同心。为君死，死亦相恋，恋亦相结、结亦相伴，伴而三生！”
<br/>
<br/>        寒伯安眼眶发热，此生何其有幸啊！
<br/>
<br/>        他拉着安男的手，走到大厅中央，招手叫来了李长卿，低声在李长卿的耳边轻声的说了几句。
<br/>
<br/>        半个小时后
<br/>
<br/>        无论是娱乐媒体还是新闻频道，都同时播放着寒伯安手持钻戒单膝跪地向安男求婚的画面。
<br/>
<br/>        “安男，你愿意嫁给我吗？此生此世相守，三生三世相恋？”
<br/>
<br/>        “我愿意！此生此世相守，三生三世相恋！”
<br/>
<br/>        寒伯安这算是彻底的公开了他和安男的关系，此生此世他们风雨同舟，相依相守！
<br/>
<br/>        陈虞和落姸在北京的家里，大声的喊着：“妈妈，快来看，寒叔叔向安阿姨求婚了。”
<br/>
<br/>        陈子昂从房间里跑出来，看得热泪盈眶：“真好！真好！”
