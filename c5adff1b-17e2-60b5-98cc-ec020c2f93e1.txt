        &lt;h3&gt;第三卷第三十一章  江俞琪的西城规划&lt;/h3&gt;
<br/>
<br/>        江家的房产事业蒸蒸日上，既某江国际社区的开发，江俞琪又将眼光投在了某江国际社区周边的地方。
<br/>
<br/>        这一片都是城中村，早在秦朝的时候，这一片就已经建有皇家禁苑，附近还有秦二世胡亥的墓陵，到了汉代，汉武帝见国库充盈，就开始倒腾，“疯狂”扩建上林苑，新建了包括宜春下苑在内的大批宫苑。
<br/>
<br/>        唐代长安城沿袭了隋大兴城的基本格局，这个城中村这块地儿大致处在修行坊、修政坊、青龙坊的范围内，在长安城的东南角角上。唐代有名的“海上生明月，天涯共此时”的诗人在这个村子住过。
<br/>
<br/>        唐代覆灭后，这个村方圆都荒芜寂寥。宋人张礼在《游城南记》是这样描写的：
<br/>
<br/>        “倚塔下瞰曲江宫殿，乐游燕喜之地，皆为野草，不觉有黍离麦秀之感。”
<br/>
<br/>        到了明清，这块地被称作“古北池”，明崇祯年间村门石匾和清顺治堡门洞石匾，上面都刻有“古北池”的名称。
<br/>
<br/>        历史变迁，经过时代更迭，这一片都是小三四层左右的民房，这里的村民世代务农，江家开始初建的两个小区除了安置工程之外，商品房已经卖完了，周边的开发也在进行着。
<br/>
<br/>        江家开发的楼盘除了江某国际集团外，还有某江铂宫，某江铂宫完全就是别墅区，几个楼盘一起开工，轰轰烈烈的进军西部开发项目在紧密锣鼓的进行着。房价从四五千到上万，人们的生活水平提高了，相对应的住房需求也提高了，江家打造的精品社区迎合了大部分人的消费观念，人们开始注重居住环境，开始看重附近的教育及周边的配套设施。
<br/>
<br/>        江家参与到了这一片社区的改造之中。根据区域规划，他们将要在这一个区域建立大型的社区，包括学校。
<br/>
<br/>        江俞轩虽然没有参与到具体的事务中，但是每一个项目他都知情，在他的职业生涯中，他不但是陈子昂公司的副总，他还是江家大少。
<br/>
<br/>        江家有意识磨练江俞轩，对他在外面的事情也是一直关注着，而且家里的事情也是会让他知情，并且参与到决策中来。
<br/>
<br/>        尤其是最近，全国各地都在大规模的拆迁新建，江家需要拓展业务，更是需要人才，做为江家大少，等于就是不务正业了。
<br/>
<br/>        放出笼子里的鸟儿是很难收回来的，这个是江爸爸的心得，那个时候想着让江俞轩出去磨练磨练也很好，没有想到这一磨练就野了性子，叫不回来的，不过看着几个年轻人在外面做的风生水起，他就觉得让回来的理由也不太充分。
<br/>
<br/>        于是江爸爸就准备采取迂回战术，让姐姐经常打电话催促江俞轩回去。
<br/>
<br/>        江爸爸和江俞琪都是在不同的地方工作，平时也都电话联系，当然他们也是到处飞的，每一个地方都会经常去视察一番。
<br/>
<br/>        江爸爸站在广州总部的大楼上看着外面的艳阳天，想着儿子什么时候回来，他也很久没有见到儿子了。
<br/>
<br/>        于是给江俞琪发信息：
<br/>
<br/>        爱豆：（江俞琪对老爸的爱称）“琪琪，最近有和轩轩联系吗？”
<br/>
<br/>        安琪儿：“轩轩这两天在西省一个下属的县城呢。”
<br/>
<br/>        “他们在干什么？”
<br/>
<br/>        “爸，我觉得你很奇怪呢，每次你想轩轩了都会问我，为什么你自己不问他呢？”
<br/>
<br/>        “那臭小子很久都没有给我打电话了。”
<br/>
<br/>        “那你给他打电话啊。”
<br/>
<br/>        “我才不给他打电话呢。”
<br/>
<br/>        “好了，好了，我让他给你电话吧，不过这几天他可能很忙。”
<br/>
<br/>        “他们最近在忙些啥？”
<br/>
<br/>        “听伯安说他们最近合作着搞一个新兴的项目，另外他们自己的公司也在搞研发，爸，陈子昂将她名下的股份都给一些高层平均分配了。”
<br/>
<br/>        “哦？”
<br/>
<br/>        “轩轩是他们公司的第三大股东。”
<br/>
<br/>        “哈哈，这小子这几年没有白熬啊。”
<br/>
<br/>        “江大佬还能看上那一点股份？”
<br/>
<br/>        “那一点股份对我们来说，确实不算什么，但是确体现了俞轩的价值，毕竟他们公司起步时间不长，能做到这一地步已经很不容易了。”
<br/>
<br/>        “爸爸，这是在夸自己的儿子吗？”
<br/>
<br/>        “不，我夸他们公司的所有人，他们的成就是一个团队的成就，而不是某一个人，固然领导人很重要，但是要知道独木不成林。”
<br/>
<br/>        “爸爸，要不要来西城，看一下我们最近的项目？”
<br/>
<br/>        “最近碰到瓶颈了吗？”
<br/>
<br/>        “这倒没有，只是之前停了两天工？”
<br/>
<br/>        “怎么回事？”
<br/>
<br/>        “大概是有人看我们不爽，举报我们没有经过勘查，说是我们开发的地盘下面有东西。不过现在已经解决了。”
<br/>
<br/>        “怎么解决的？”
<br/>
<br/>        “陈子昂的二哥刚好是那个单位的老总。”
<br/>
<br/>        “轩轩，准备还要在他们公司呆多久？”
<br/>
<br/>        “轩轩说先结婚，他们目前还要研发一个新产品，等新产品上市了他就回来。”
<br/>
<br/>        “得多久？”
<br/>
<br/>        “少则一年，多则两到三年。”
<br/>
<br/>        “西城的项目让他参加吧！”
<br/>
<br/>        “我怕他没有精力，他们公司最近确实也很忙，到处在拓展业务，而且和寒伯安，郝景文他们又合作了一个项目。”
<br/>
<br/>        “西城的项目很大，目前的状况只是一小部分，你现在所处的片区都在改造。我知道他们三个人合作的项目，目前趋势还不错，但是不妨碍他参加我们的公司的事情。”
<br/>
<br/>        “好，我给轩轩说，不过，他们公司在西城也有分公司，也许他能兼顾的到。”
<br/>
<br/>        “让他参与，但是没有必要让他亲力亲为。”
<br/>
<br/>        “明白。”
<br/>
<br/>        “好了，我这里还有事情，再见！”
<br/>
<br/>        “爸爸，再见！”
<br/>
<br/>        江俞琪和父亲结束了聊天，坐着发了一会愣。西城的项目目前有三个，已经完工的有一个，再建的有两个，他们的计划是在十年内拿到十个项目。
<br/>
<br/>        每个项目的工期大概需要三到四年，如果江俞轩也要参与这个项目的话，他最近这七八年的时间将要经常到西城来。
<br/>
<br/>        江俞琪拨通了江俞轩的电话，江俞轩正在和傅海生聊着项目上的事情，她收到了一个短信：“现在正忙，稍后回你电话。”
<br/>
<br/>        江俞琪苦笑了一下，当初弟弟毕业的时候说是出去实习，谁知道这一实习就是六七年？不过他们家里还是很开明的，要不然也不会让他在外面浪这么久。
<br/>
<br/>        寒伯安和郝景文、江俞轩三兄弟又开展了新的业务，三个家长其实都是知道了，用三个家长的话说：“不乘着大好年华干一番事业，岂不是浪费了那激情澎湃的岁月？”
<br/>
<br/>        江家在西城的房地产业象征着一个时代的变化，在今后的几年里，他们都见证了西城的变化。
