　　武道碑林内，三百武者几乎都找到了适合自己参悟的玄界碑。
<br/>　　只不过，真正能开始参悟的，却寥寥无几。
<br/>　　不过数分钟，三百人，已有近百人因承受不住而晕厥倒地。
<br/>　　剩下的两百多人，也大多在苦撑着。
<br/>　　碑林外，十位内门执事暗暗点头。
<br/>　　“今年的弟子很不错，起码在坚韧程度这一点，超出了我们的预期。”
<br/>　　“不错，若无足够的领悟力，强行去参悟玄界碑，只会自讨苦吃。但若是心智足够坚定，倒还是能有所收获的。”
<br/>　　十位内门执事，作为这第三个考核的‘考官’，正做着公允的评价。
<br/>　　“你们看，有人开始领悟了。”
<br/>　　十人目光一亮，纷纷看向开始领悟之人。
<br/>　　那高达数十米的玄界碑，散发着一股古朴且锋锐无双的气势。
<br/>　　碑下，盘膝坐着一少年，正是萧逸。
<br/>　　“不错，观此子的年纪，不过16、7，却有先天九重修为，在武道上的天赋，足可列入超级天才的行列。”
<br/>　　“也不知这少年在炼药一道上可有天赋，否则，倒是可以加入我药堂。”说话的这位执事，来自于药堂。
<br/>　　“你不用多想了。”剑堂的执事笑道，“没看到吗，他参悟的是剑道的玄界碑，肯定是要加入我剑堂的。”
<br/>　　另外八堂的执事也满意地笑了笑，“能如此迅速地承受住玄界碑禁制的影响，证明此子的领悟力惊人卓绝，无论修炼哪种武道，成就都定然不低。”
<br/>　　“无论他加入哪一堂，日后都必是名震北山郡的天才。可惜了，他参悟的是剑道玄界碑，看来他是想加入剑堂的。”
<br/>　　“哼，你们就眼馋吧。”剑堂执事笑道，“此子注定会加入剑堂。”
<br/>　　....
<br/>　　另一边，玄界碑下的萧逸，并不知执事们正评论着他。
<br/>　　他只是在尽自己的能力，快速地消化着碑上的内容。
<br/>　　一块玄界碑，便是一位破玄境武者一生的修炼知识。
<br/>　　一个个奇怪的文字，一道道玄妙的画面，正不断在他脑海中闪过。
<br/>　　所谓参悟玄界碑，乃是玄界碑通过‘醍醐灌顶’的方式，将这些知识传入参悟者的脑海。
<br/>　　而这些知识，远远超过了先天境武者所能理解和承受的范畴。
<br/>　　这也是为什么其他武者会头晕目眩，甚至晕厥过去。
<br/>　　领悟力高的，便能接受这些内容，理解了，脑海中就会出现新的文字和画面。
<br/>　　领悟力低的，不能接受，却强行去观看，脑海中出现的画面，便会定格下来，承受不住，就会心神接近奔溃，头痛欲裂，难受至极。
<br/>　　“额。”萧逸咬了咬牙，脸上忽然涌现了难受的表情。
<br/>　　那些文字和画面，就是这位剑道长老一生的修炼知识。
<br/>　　他能领悟，但却也需要领悟过程。而在这过程中，在未完全领悟之前，他还是会承受那种疼痛的折磨。
<br/>　　只有完全领悟，阔然开朗，折磨才会消失。
<br/>　　可是，领悟完这一副画面，又会出现新的画面，新的文字。
<br/>　　这些新的画面，会再次让他陷入痛苦折磨中。
<br/>　　萧逸的表情，也不断在痛苦和释然中不断变换着。
<br/>　　他已经数不清自己脑海中闪过多少画面，又领悟了多少文字了。
<br/>　　只知道，面前巨大玄界碑上的内容，他领悟了三分之一左右。
<br/>　　第三轮考核，考验的是领悟力、忍受力、心智是否坚定等等。
<br/>　　而这些，都属于天赋，也是一个真正的武道天才应该具备的表现。
<br/>　　这才是第三个考核存在的意义。
<br/>　　大半个时辰后，武道碑林内，一切都成了定局。
<br/>　　能领悟的人，已经开始在玄界碑下参悟。
<br/>　　不能领悟的人，就算强撑，也撑不了多长时间。
<br/>　　半个时辰的折磨，足以让他们彻底承受不住，晕厥过去。
<br/>　　内门执事们大手一挥，操控天地灵气，凭空将碑林内晕厥的人拉了出来。
<br/>　　还在碑林内的，还剩下二十人。
<br/>　　“如无意外，今年就是这二十人能进入内门十堂了。”执事们说道。
<br/>　　每年参加考核的武者，都有过万人，而能来到第三轮考核的，只有三百人，真正通过的，也只有十人左右。
<br/>　　也难怪内门弟子如此少，但待遇却极高。
<br/>　　今年能有二十人，已经远超以前了。
<br/>　　“看，那边的少年也不错，领悟的是刀道的玄界碑。我没记错的话，他是第二个能稳住心神，进入参悟状态的吧。”
<br/>　　一位执事看向了碑林中的林劲。
<br/>　　刀堂执事满意地点点头，“嗯，此子先天八重修为，领悟力也极高，进入我刀堂最合适不过了。”
<br/>　　“又有一人进入参悟状态了。”
<br/>　　一执事看向了另一边的铁牛。
<br/>　　兽堂执事也满地笑了笑，“先天八重修为，领悟力高，最难能可贵的，此子样貌憨厚，神色安然，似有一颗赤子之心，就入我兽堂吧。”
<br/>　　“那少女也不错。”忽然，万木堂的执事看向了柳嫣然，笑道，“她参悟的乃是植物一系的玄界碑，入我万木堂吧。”
<br/>　　万木堂的执事，是一位中年女子，风度过人，话语不多，却透露着一股让人如沐春风般的惬意气息。
<br/>　　“咦。”忽然，兽堂的执事看向了碑林中的秦飞扬，“此子参悟的玄界碑，乃是我兽堂的一位长老先人留下的修炼知识。”
<br/>　　“看来，此子也是要加入我兽堂的。今年，竟能收到两个超级天才入兽堂，啧啧，真是幸运。”
<br/>　　“.....”
<br/>　　十位内门执事，已经通过碑林中剩下的二十人参悟的玄界碑类型，而确定了他们进入哪一堂。
<br/>　　内门十堂，便是内门弟子修炼和居住的地方。
<br/>　　时间飞逝，一个时辰后，碑林内的情况，再次发生了变化。
<br/>　　“噗。”顾长风猛地从参悟中醒来，吐出一口腥血。
<br/>　　“噗。”花剑、王虎等人也吐了一口腥血。
<br/>　　....
<br/>　　足有十数人，从参悟状态中醒来，口吐鲜血，脸色苍白。
<br/>　　但十位执事，却没有丝毫的不满，反而满脸笑容。
<br/>　　“能参悟一个多时辰，这十数弟子已经相当不错了。虽未能完全悟透整块石碑，却起码在四分之一以上。”
<br/>　　“往年参加这项考核的弟子，大多只能参悟半个时辰左右，参悟玄界碑十分之一左右的内容。”
<br/>　　“这十数弟子，天赋已经很了不起了。”
<br/>　　十位执事赞扬着，并大手一挥，将这十数人拉了出来。
<br/>　　碑林内，还剩下无人，正是萧逸，林劲、铁牛、柳嫣然和秦飞扬。
<br/>　　又过了两个时辰，林劲等四人相继从参悟中醒来。
<br/>　　“噗。”林劲吐了一口腥血，脸色煞白，遗憾道，“可惜了，整块玄界碑，领悟的内容越是多，越到后面，难度就越大。”
<br/>　　“足足参悟了三个多时辰，却只参悟了一半多点。”
<br/>　　秦飞扬、铁牛和柳嫣然，脸上也是遗憾的神色。
<br/>　　殊不知，碑林外的十位内门执事，早已满脸惊骇之色。
<br/>　　“这四人，竟在玄界碑的禁制下参悟了三个多时辰，承受那种非人的折磨这么久，当真厉害。”
<br/>　　“他们如今不过先天境，却已经参悟了玄界碑上大半的内容，日后的武道之路，必定一帆风顺，一日千里。”
<br/>　　执事们赞叹着，大手一挥，将这四人也拉出了碑林。
<br/>　　下一秒，十位执事，特别是剑堂执事，目光灼灼盯着碑林中如今仅剩的唯一一道身影。
<br/>　　“那少年还在参悟着。”
<br/>　　“他是第一个进入参悟状态的，却也是坚持最久之人。”
<br/>　　“也不知他能走到哪一步。”
<br/>　　执事们充满了期待。
<br/>　　碑林中，萧逸仍旧参悟着。
<br/>　　事实上，他和林劲等人参悟的时间差不多，但参悟的速度，却远超他们。
<br/>　　如今，整块玄界碑上的内容，他几乎都参悟完毕了。
<br/>　　只剩下最后一丁点儿。
<br/>　　而这一丁点儿内容，记录了一门武技，名为‘电光火石’，玄阶巅峰。
<br/>　　玄界碑上，并没有地阶武技，最高只到玄阶武技。
<br/>　　并非是这些破玄境的长老没有地阶武技，而是特地没有记录到玄界碑上罢了。
<br/>　　地阶武技，哪怕是低级，也比玄阶巅峰武技珍贵百倍，自然不会随便给予。真正的典籍，放置在内门宝库中，轻易无法得到。
<br/>　　当然了，除了地阶武技外，这位剑道长老的一生修炼知识，包括洞玄境、破玄境的修炼奥妙，都记录在玄界碑上了。
<br/>　　萧逸认为，这些才是真正的财富，也是他日后武道修炼最重要的东西，珍贵无比。
<br/>　　萧逸领悟得更快，却也承受了更多的折磨。
<br/>　　那不断闪过的画面和文字，让他承受折磨的次数，远多于林劲等人。
<br/>　　这种侵蚀心神的折磨，最是疼痛。
<br/>　　而且，如果是一直痛下去也就罢了；偏偏领悟过程疼痛，领悟完毕释然；这种一时地狱，一时天堂的感觉，最是磨人。
<br/>　　他的脸色，惨白得吓人；额头上，布满了豆粒大的汗珠；汗水，甚至打湿了整幅衣衫。
<br/>　　与其说他的天赋比林劲等人强，倒不如说他的内心比任何人都要坚定，都要坚韧不拔。
<br/>　　当然，他苦苦支撑承受，为了那些珍贵的武道知识，还有武技‘电光火石’，却甘之如饴。
<br/>　　电光火石虽是玄阶巅峰武技，但它的威力，却可以做到碾压其它任何玄阶巅峰武技。
<br/>　　被誉为裂天剑派最强玄阶巅峰武技，甚至可媲美地阶低级武技。
<br/>　　如无意外，这将是萧逸至今为止拥有的最强武技。
