　　“老祖！”
<br/>　　尹寒月看到这老者，顿时大喜，不由得惊呼了一声。
<br/>　　原来这位老者，便是飘香苑背后的那位半步化神强者了。
<br/>　　周围众人也纷纷惊呼起来，
<br/>　　“这位就是尹家的那位老祖吗？”
<br/>　　“不愧是半步化神境啊，果然恐怖！”
<br/>　　“连尹家老祖都出来了，这下总应该可以阻止那小子了吧？”
<br/>　　……
<br/>　　众人纷纷议论着。
<br/>　　然而，
<br/>　　叶尘却眼皮都没有抬一下，双眸中透着一抹杀意，冷冷道：
<br/>　　“老头，你也想找死吗？”
<br/>　　轰！
<br/>　　叶尘这话一出口，周围顿时一下子炸开了锅。
<br/>　　“这小子是疯了吗？竟然敢和尹家老祖这般说话？”
<br/>　　“虽然这位尹家老祖只有半步化神，可也是一只脚迈入化神境的大人物，在云丹城明面上的各方势力之中，实力足以排进前二十！”
<br/>　　“化神之下，皆为蝼蚁，这小子虽然厉害，可也不该却招惹一名化神强者！”
<br/>　　“同时得罪屠苏家和尹家，这小子，完了！”
<br/>　　……
<br/>　　周围众人开始幸灾乐祸起来。
<br/>　　而尹寒月看到自己老祖出现，顿时精神一震，底气也足了许多，
<br/>　　“老祖！这小子在我们飘香苑动手打人，而且还毁坏了无数财物，您快点出手将他拿下……”
<br/>　　“闭嘴！”
<br/>　　尹寒月话还没有说完，尹家老祖忽然一色暴喝，将她后面的话硬生生堵了回去。
<br/>　　不但尹寒月懵了，周围众人看到尹家老祖突然发火，也全都纷纷安静下来。
<br/>　　然而，
<br/>　　接下来发生的一幕，却让所有人目瞪口呆。
<br/>　　只见，尹家这位半步化神境的老祖，竟然冲着那个打伤屠苏元庆的年轻人，微微躬身行了一礼，苍老的脸庞之上，还露出一抹略带讨好的笑容，
<br/>　　“小辈们不懂事，道友千万勿怪！”
<br/>　　哗！
<br/>　　所有人全都傻眼了。
<br/>　　在他们之前想来，尹家老祖一出手，必定会将这个闹事的小子一举擒下。
<br/>　　然而，任谁也没有想到，这尹家老祖不但没有动手，反而一副讨好的姿态。
<br/>　　难道这个无法无天的年轻人，有着什么恐怖的背景不成？
<br/>　　“老祖，您这是在做什么？你怎么可以……”
<br/>　　尹寒月感觉整个人都要崩溃了，
<br/>　　一向被她视作神明一般的老祖，竟然向眼前这个可恶的小子屈服了？
<br/>　　尹家老祖猛然回过头，脸一板，
<br/>　　“寒月！但凡进入我飘香苑的，都是我们的客人，你怎么可以对客人如此无礼？还不快向贵客赔礼道歉！”
<br/>　　尹寒月顿时两眼一瞪，简直怀疑是不是自己的耳朵出了问题，
<br/>　　“老祖，你在说什么呀？你知不知道他刚刚都做了什么？”
<br/>　　尹家老祖却一摆手，
<br/>　　“我让你道歉！你现在是不是连老夫的话都不听了？”
<br/>　　“我……是！”
<br/>　　尹寒月满脸的委屈之色，只好走上前，便要向叶尘道歉。
<br/>　　不料，叶尘一摆手，
<br/>　　“道歉就不必了！我们现在可以走了？”
<br/>　　尹家老祖连忙点头道：
<br/>　　“当然！请便！”
<br/>　　最终，
<br/>　　在众人惊愕目光中，叶尘带这灵韵和灵溪两姐妹离开了。
<br/>　　……
<br/>　　“老祖，你刚刚，为何要放了那小子？”
<br/>　　叶尘三人离开之后，尹寒月随着尹家老祖回到包厢之中，一脸的不甘问道。
<br/>　　尹家老祖没有说完，走到一张椅子前，缓缓坐了下去，正要开口，脸色猛然一变，
<br/>　　“噗！”
<br/>　　尹家老祖竟突然喷出了一口鲜血。
<br/>　　一旁的尹寒月顿时吓了一跳，慌忙走上前，一把扶住，
<br/>　　“老祖，您受伤了？怎么会这样？”
<br/>　　尹家老祖右手一翻，取出一枚丹药服下，又调息了好一会，脸色这才渐渐好转了许多，不过却满脸的凝重之色，望着尹寒月，道：
<br/>　　“丫头，现在你总该明白，老夫为什么要这么做了吧？”
<br/>　　尹寒月先是微微一呆，继而双眸一瞪，
<br/>　　“您的意思是说，您受伤，是因为刚刚那一掌？那小子的实力，竟然强大到如此地步？这怎么可能！”
<br/>　　尹家老祖深吸了一口气，缓缓道：
<br/>　　“我也觉得这不可能！可事实就是如此！”
<br/>　　“虽然是我太过大意，所以才会被对方的暗劲所伤，可是败了就是败了！”
<br/>　　“我刚刚若是不那么做，恐怕你的小命，此时早就已经不在了！”
<br/>　　尹寒月还是觉得有些难以接受，
<br/>　　“可是那小子，明明只有金丹境的实力，您可是半步化神境，怎么会？这这么会？”
<br/>　　尹家老祖幽幽叹了一口气，道：
<br/>　　“正因为如此，此人才更可怕！”
<br/>　　“我敢断定，此子要么是某个超级大宗门的绝顶天才，要么就是某位超级大能转世，否则不可能以区区金丹境的修为，就拥有如此实力！”
<br/>　　“而无论是哪一种情况，都不是我们区区尹家可以招惹的！”
<br/>　　说到这里，尹家老祖微微一顿，接着又道：
<br/>　　“至于屠苏家那边，我已经派人将屠苏二公子送回府中，并且将情况说明，即便今天的事情，他们会对我们的做法有所不满，却也不会真把我们怎么样！”
<br/>　　“而且，屠苏家想要为他们的二公子报仇，也未必有那么容易！”
<br/>　　“这云丹城，怕是要乱了！”
<br/>　　……
<br/>　　另外一边，
<br/>　　云丹城五大家族之一的屠苏府，此时传出一道咆哮之声，
<br/>　　“谁？到底是谁？竟然敢将我儿残害成这个样子！我要将他碎尸万段！！”
<br/>　　声音犹如天雷滚滚，连同诺大的屠苏府，都跟着一阵乱颤。
<br/>　　这咆哮之人，是一名身穿魁梧的中年男子，身上的煞气惊人，气息竟不在尹家老祖之下。
<br/>　　原来此人，便是屠苏元庆的老子，屠苏家的家主，屠苏玄！
<br/>　　而此时，在屠苏玄的身侧，还站着一名满脸傲然之气青年男子，如果叶尘在这里的话，便能一眼认出，此人正是在考核大会上见过的那位屠苏公子。
<br/>　　原来他竟是屠苏元庆的哥哥，名为屠苏元让！
<br/>　　“我二弟在飘香苑，被人残害成这个样子，你们飘香苑就不管不问吗？尹寒月是干什么吃的？”
<br/>　　屠苏元让望着飘香苑的使者，冷冷开口道。
<br/>　　那名使者顿时吓得浑身一哆嗦，慌忙跪俯在地上道：
<br/>　　“启禀大公子，我家老板原本也想将凶手拿下，然后交给贵府处置，奈何那行凶者实力太强，最终还是让他给逃了，不过我们已经将行凶者的容貌画了下来！”
<br/>　　那人说着，右手一翻，从储物袋中，取出了一副画像，然后缓缓打开。
