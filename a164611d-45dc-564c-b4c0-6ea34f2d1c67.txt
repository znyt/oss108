　　“有关丧尸引诱剂……”何峰的表情有了一丝凝重，看向陈少爷的目光很严肃：“射击队的人手有限，更不能出任何差错，所以丧尸引诱剂的引爆只有你一个人。”
<br/>　　屋内顿时一静，很多人都面色变化起来，陈少爷虽然是命中率近乎百分百的神之狙击，但这次大战光是人类的士兵参与者就高达九万，战区更是含盖了整个南署基地，可想而知战役规模之巨大，任务又何等艰巨。
<br/>　　陈少爷一个人，能行吗？
<br/>　　最重要的是，他能兼顾这么多个方位的丧尸引诱剂引爆吗？
<br/>　　现场的气氛虽然凝重，很多人都目光带上了担忧，但陈少爷却还是那副憨笑的样子，道：“我一个人也行，但最好给我配一架直升机，这样方便些，我毕竟速度不行。”
<br/>　　听着陈少爷的话，众人都开始翻起了白眼，路冰泽更是无语的吐槽道：“你特么想的倒是挺好！”
<br/>　　但让众人没想到的是，此时的何峰却点了点头：“好，发射器还有丧尸引诱剂的子弹都已经加以改良，现在全部放在了一架直升机中，有专门的驾驶员，你只需要记住每个点的位置然后射击就行，但是我必须要提醒你一点，直升机内的引诱剂固态子弹数量有限，每一次发射都不能有失误，一轮完毕后再回基地的最中心地补给，然后继续发射，也是持久战。”
<br/>　　何峰一口气说完，信息量又多又让人出乎意料，屋内再一次安静的针落可闻。
<br/>　　众中将都面面相觑，更有甚者目瞪口呆的看着何峰和陈少爷，完全被镇住了。
<br/>　　“这直接让直升机给陈少爷当座驾的架势，真是够壮观的啊！”路冰泽有些羡慕的开口。
<br/>　　“还有更壮观的呢！”何峰白了他一眼，紧接着面色在此变得严肃：“最后一点，这次尸潮会有飞行丧尸出现，虽是猜测，但推断而出的几率高达90%。”
<br/>　　这下子路冰泽面色一下子从羡慕变成了暗沉，飞行丧尸他们可是见过的，尤其是三战队对此可是异常熟悉，倾巢大战之前就是因为那只飞行丧尸叼走了楚涵，导致独自前往城市寻找楚涵的黑芒战队与大部队分开，让神秘家族的那批队伍有机可乘，半路截杀给了落单的黑芒战队一次重击。
<br/>　　那是黑芒成立至今最大的重创，过半的黑芒战队成员阵亡，余下的成员人人重伤，最终以黑芒战队的队长肖坤战死，黑芒战队差点解散的可怕程度收尾。
<br/>　　索性楚涵的及时出现，二十名黑芒战队老队员的出世闯荡，中途江凌若的强势加入。
<br/>　　才让黑芒战队复活，并一举重振！
<br/>　　但肖坤的战死没有人能放得下，更是狼牙战团所有老兵心中的一根刺，而导致这种心理形成的最关键点，就是飞行丧尸。
<br/>　　他们见识过，印象极其深刻，永生难忘！
<br/>　　没有人说话，路冰泽也不再调笑，所有人的表情都很一致，那就是严肃中带着坚定。
<br/>　　狼牙上回没能杀死那头飞行丧尸，甚至差点败在那里，这是一道必须越过去的砍，也是哪怕路冰泽绞尽脑汁突破了种种瓶颈之后，也无法解决的一大难题。
<br/>　　飞在天上的，他们怎么杀？
<br/>　　何峰这时候忽然眼神转移，看向了军法处长刘玉定：“交给你们军法处。”
<br/>　　“啊？”刘玉定愣住，下意识的一声惊疑。
<br/>　　这是他胜任军法处长之后极少会出现的神色，说明不仅没有料到，更完全不能理解也没有把握。
<br/>　　其余人更是误以为自己听错了，军法处的人当然也在战区，但他们是随着那五万新兵的训练跟过来的，说白了并不是真正的战斗型成员，虽然很多军法处的老成员因为杀羽战队不怎么服从惩罚的原因，开始了新人类之旅并步步提升等阶。
<br/>　　但是他们是文官啊！
<br/>　　论战斗根本就是弱项，而且还是飞行丧尸的战斗，这不是开玩笑么这？
<br/>　　反应慢半拍的陈少爷掏了掏耳朵，眨着眼睛看着众人，表示自己除了射击队的任务之外，大部分都没听懂，而军法处的这则奇怪任务，则是完全不懂。
<br/>　　何峰没理会众人的呆滞目光，又转头看向了陈少爷，道：“射击队清点一百人出来，并入空军。”
<br/>　　“啊？”这回换成了陈少爷一声惊疑，而后目瞪口呆。
<br/>　　何峰坚毅的轮廓勾起了一个笑容，这个笑容有些诡异，但也是狼牙高层万分熟悉的笑容。
<br/>　　因为他们经常在楚涵的脸上见到！
<br/>　　这个笑容里透露着奸诈、自信还有狂妄……
<br/>　　“我们有科技部新发明的一百架战斗直升机，专门对付飞行丧尸！”何峰开口道。
<br/>　　哗！
<br/>　　屋内全体人都哗然而起，他们激动又震撼，差点就要吼叫出来。
<br/>　　除了最先来到战区并见过那批直升机的杨天有心理准备，但此时听到何峰这么说，还是表情瞬息万变了一下，道：“那百来架外貌奇怪的直升机，果然就是战斗机？”
<br/>　　何峰点头：“已经得到和上的证实，并且也是楚涵上将这次战略中的一环。”
<br/>　　杨天嘴角狂·抽了好几下，语态都变了：“你们竟然让直升机拉货？浪费可耻啊我靠！”
<br/>　　这下子一说，所有人都明白了过来，军法处的人自然没办法待在第一战线与丧尸正面战斗，但是他们也都是军队出身，狼牙军规更是他们一手制定，对狼牙战团的每一方面都熟悉至极，全面到同时包括如何架势直升机。
<br/>　　狼牙的人向来全面发展，因为很多时候都人手不足，不得不各部门的人综合。
<br/>　　于是这会儿的刘玉定终于明白了过来，长舒一口气道：“原来只是架势直升机啊，早说啊，吓死我了！”
<br/>　　让军法处的人开个直升机还是没问题的，哪怕这是他们第一次正式参战。
<br/>　　“军法处的人也是士兵，我们不知道尸潮的规模，当然能用上的就用。”何峰笑了笑，道：“驾驶员全体又军法处担任，趁着这几天大战还没开始，你们的人赶紧先熟悉下。”
<br/>　　“射击队的一百人，一定要精心挑选。”何峰再次看向陈少爷：“飞行丧尸本就不好瞄准，你又还有其他任务，这是我们的第一次空战！”
